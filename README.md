# INT332: DevOps Virtualization and Configuration Management

**Author:** Mohammed Hamza  
**Repository:** [Hamza8470/INT332-Devops](https://github.com/Hamza8470/INT332-Devops)

This repository contains the theory notes, practical commands, and coursework tasks for the INT332 DevOps course. It covers containerization, Docker, microservices, and Maven build automation.

## Course Outcomes
* **CO1:** Explain DevOps infrastructure concepts including containerization, runtimes, process isolation, namespaces, and resource management mechanisms.
* **CO2:** Apply Docker concepts to build, manage, and distribute container images using networking, storage, and registries.
* **CO3:** Construct microservices applications using Docker Compose with multi-container services and dependency configuration.

## Directory Structure
* **[Unit I: Basics of DevOps Infrastructure](./Unit-1-Basics-of-DevOps-Infrastructure/)** - Introduction to containers, Docker architecture, and fundamental CLI commands.
* **[Unit II: Image Building & Container Management](./Unit-2-Image-Building-Container-Management/)** - Dockerfiles, networking, volumes, and registries.
* **[Unit III: Microservices with Docker Compose](./Unit-3-Microservices-Docker-Compose/)** - Multi-container applications, YAML configuration, and deploying stacks.
* **[Unit IV: Maven Build Automation](./Unit-4-Maven-Build-Automation/)** - Build lifecycles, POM, dependency management, and Dockerizing Maven apps.
